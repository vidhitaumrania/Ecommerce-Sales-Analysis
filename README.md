# 🛒 Ecommerce Sales Analysis

## 📌 Overview
The **Ecommerce Sales Analysis** project focuses on analyzing ecommerce sales data to discover meaningful business insights and trends. This project helps businesses understand customer behavior, product performance, sales growth, and profitability through data analysis and visualization techniques.

Using Python and data analysis libraries, the project performs data cleaning, exploratory data analysis (EDA), and visualization to support better business decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-selling products and categories
- Understand customer purchasing behavior
- Evaluate profit and discount impact
- Visualize sales trends and patterns
- Generate actionable business insights

---
